Servo dashboards

Servo dashboards for performance and build times.

Build times

Build times are maintained by buildbot, and downloaded using the JSON API by a script that is run nightly. The data is stored on S3, and imported into Google Data Studio.

T5n performance

Servo nightly builds are run against the Talos TP5 test suite by CI scripts. The data is stored on S3, and imported into Google Data Studio.

Archived web performance

Servo nightly builds are run against a collection of archived web content by CI script. The data is stored on S3, and imported into Google Data Studio.
